1. Tạo 1 Task trên hệ thống, xem thử nó có mã Task Key là gì nhé (Ví dụ: PRJ-1).
2. Quay lại giao diện máy sinh để code. Vào thư mục gốc của cái Repository đó (trong terminal hoặc Vscode).
3. Thêm một Commit rỗng hoặc sửa 1 đoạn chữ nhỏ xíu rồi Commit thực tế: git commit -m "Test tự động hóa luồng DevOps cho PRJ-1"
4. Gõ: git push
